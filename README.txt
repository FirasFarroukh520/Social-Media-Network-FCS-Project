Social Media Network FCS Project
This project implements a simple social network using Python and the NetworkX library. It includes functionalities for adding users, creating relationships, calculating centrality measures, and visualizing the social network graph.

Features: 
Add and remove users
Add and remove relationships between users
Calculate shortest paths between users
Calculate degree and betweenness centrality
Visualize the social network graph using Matplotlib

Prerequisites:
Before you begin, ensure you have the following installed on your local machine:

Python 3.x
pip (Python package installer)

Installation:
Clone the repository: using cmd or vs code console:
git clone https://github.com/yourusername/social-network-project.git
cd social-network-project

Install required packages: using cmd or vs code console:
pip install -r requirements.txt
